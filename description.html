Description de votre projet

Par Biollaz Benjamin, Gaillard Nathan et De Blasi Simon le 31/03/2022 à 14:45

Navigateur solitaire, vous entrez sans le savoir en territoire hostile. Les formes de vies locales ne semble pas vouloir discuter. A peine arrivé, votre vaisseau subit des attaques. Une seule solution : servez-vous des armes de votre vaisseau pour éliminer l'ennemi !

Les vagues d'ennemis deviennent de plus en plus nombreuses, combien pouvez-vous en vaincre ?

Le code source de l'application est disponible sur le github du projet.
